Module 0: Gearing Up for the Adventure

Objective: Set up your programming environment and run your first Rust program.

Welcome to your first step! In this module, we'll get all the tools you need and make the computer say "Hello" in Rust.

Step 1: The Grand Welcome

What is Rust? Imagine you're building with super-powered LEGOs. Rust is like a wise and friendly instructor who checks every piece you add, making sure your creation won't fall apart. It helps you build things that are both incredibly fast and super safe.

Step 2: Setting Up Your Workshop

We need to install the Rust toolchain. The best way is to use a tool called rustup. Visit the official Rust website at rust-lang.org and follow the instructions for your operating system (Windows, macOS, or Linux). This will install everything you need, including:

  • rustc: The Rust compiler that turns your code into an executable program.
  • cargo: Rust's amazing project manager and build tool. Your new best friend!
  • rustfmt: A tool to automatically format your code to keep it neat.

Step 3: Our First Command: cargo new

cargo is like a magical toolbox that creates projects for you. Let's make one!

Open your terminal or command prompt and run this command:

cargo new todo_list

This creates a new folder called todo_list. Inside, you'll find two important things:

  • Cargo.toml: The project configuration file.
  • src/main.rs: The file where we'll write our Rust code.

Step 4: "Hello, Adventurer!" - Your First Program

Open the src/main.rs file in a text editor. It already has some code!

fn main() {
    println!("Hello, world!");
}

Let's break it down:

  • fn main() { ... }: This defines the main function. Every adventure needs a starting point, and this is it for every Rust program.
  • println!("Hello, world!"): This is a macro that prints text to the screen. We know it's a macro because of the exclamation mark !.
  • ;: The semicolon marks the end of the statement.

Now, let's run it. Navigate into your new project directory in your terminal and run:

cd todo_list
cargo run

You should see Hello, world! printed on your screen. Congratulations, you are officially a Rust programmer!
